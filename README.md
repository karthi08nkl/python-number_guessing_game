# Number Guessing Game

A beginner-friendly **Number Guessing Game** built in Python where players attempt to guess a randomly generated number between **1 and 100**. The game features multiple difficulty modes, input validation, hints after each guess, and win/lose outcomes based on the selected mode.

## Features

* Random number generation
* Three difficulty modes:

  * **Easy** – Unlimited attempts
  * **Moderate** – 10 attempts
  * **Hard** – 5 attempts
* Hints after every guess (**HIGH** or **LOW**)
* Input validation
* Displays the total number of attempts used
* Announces win or loss at the end of the game
* Interactive command-line interface

## Technologies Used

* Python 3

## Python Concepts Used

* Functions
* Variables and Data Types
* Loops (`while`)
* Conditional Statements (`if`, `elif`, `else`)
* Pattern Matching (`match-case`)
* Exception Handling (`try-except`)
* Random Module
* User Input Validation
